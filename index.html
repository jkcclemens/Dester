Dester

An incoherent IRC bot that generates fun.

Dester is an IRC bot that logs messages and plays them back. But he doesn't do it like you think.

Dester reads in every message that he receives in the channels that he resides in. He doesn't record when it was sent, who sent it, or anything else; he just records the content.

Later, if anyone mentions his name, he'll respond by grabbing two random lines that he's recorded, splicing them together, and sending it to the person that mentioned him.

Dester's pretty easy to talk to, even if he doesn't make much sense. He gets better as people around him chat more.

Learn More

If you're interested in testing out Dester, he's live in the #dester channel on EsperNet. Just say his name, and he'll respond!

Code Specifics

Dester is written in the programming language called D. It is a flexible language with C-style syntax (very similar to Java) that can be both compiled and interpreted. When compiled, it generates native machine code (binaries). D produces larger binaries than C, but they can be optimized, stripped, and compressed to much smaller sizes.

Dester uses no external frameworks or libraries. The IRC protocol is handled with homegrown code, allowing for the fastest and smallest binaries.
